## 财务指标

-   接口用途  
     获取个股财务指标，包括每股指标、盈利能力、成长能力、营运能力、偿债及资本结构、现金流量

    ```javascript
    dip.stock.finance.getGuideLine("000651", "2018").then((data) => {
        //数据存储、处理逻辑，请自行实现
    });
    ```

## 资产负债表

-   接口用途  
     获取个股资产负债表，根据行业划分，包括 4 类返回结果：银行、保险、证券、普通工商业
-   接口调用说明
    ```javascript
    dip.stock.finance.getBalanceSheet("000651", "2018", "general").then((data) => {
        //数据存储、处理逻辑，请自行实现
    });
    ```

      </table>
